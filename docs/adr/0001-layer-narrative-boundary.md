# ADR 0001: 教程分层叙事与仓库边界对齐

- Status: Accepted
- Date: 2026-03-03
- Related: #3, #4

## Context

教程仓库需要保证：

1. 内容叙事严格对应 L1-L5 分层
2. 不引导用户使用过时或越层运行路径
3. 教程示例不承担生产兼容层职责

## Decision

- 在仓库根 README 明确引用边界文档与 ADR
- 统一将教程职责限定为“教学与示例”
- 对示例说明中的层级描述进行修正（L4 示例标注修复）

## Consequences

- 学习路径更一致，可减少跨层误导
- 后续新增示例需要先满足边界清单，再补充到教程目录
